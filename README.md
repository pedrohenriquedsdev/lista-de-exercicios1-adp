# 🧠 Exercícios de Lógica de Programação

Este repositório reúne **23 exercícios de lógica de programação** que desenvolvi durante meus estudos.  
O objetivo foi **fortalecer o raciocínio lógico, entender estruturas básicas da programação e praticar resolução de problemas**.

## 🚀 Sobre os exercícios

Os exercícios abordam conceitos fundamentais presentes em praticamente qualquer linguagem de programação:

- Entrada e saída de dados
- Operações matemáticas
- Estruturas condicionais (`if / else`)
- Estruturas de repetição (`for`, `while`)
- Organização do código
- Raciocínio lógico para resolução de problemas

Cada exercício foi pensado para **treinar a forma de pensar como programador**, não apenas escrever código.

## 📚 O que você vai encontrar

Entre os desafios resolvidos estão:

- Cálculo de volumes e consumos
- Manipulação de números
- Estruturas condicionais
- Laços de repetição
- Problemas clássicos de lógica (como **FizzBuzz**)

Todos os exercícios foram implementados com foco em **clareza, simplicidade e aprendizado**.

## 🎯 Objetivo do projeto

Este projeto faz parte da minha jornada aprendendo programação.  
A ideia é **documentar o progresso, praticar lógica diariamente e construir uma base sólida** antes de avançar para problemas mais complexos.

## 💻 Tecnologias utilizadas

- C#
- .NET
- Console Application
